# Passport Photo App By WEBGUNO

This repository contains the Passport Photo App by WEBGUNO. Run it locally to create and export compliant passport, visa, and ID photos.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Open the local development URL shown in your terminal (usually `http://localhost:5173`).
